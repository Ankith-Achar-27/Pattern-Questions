# Pattern Questions in Java

A collection of classic pattern programming problems implemented in Java using nested loops. This repository is intended for beginners learning control flow, loops, and problem-solving techniques.

## Topics Covered

- Square Patterns
- Right Triangle Patterns
- Inverted Triangle Patterns
- Pyramid Patterns
- Hollow Patterns
- Diamond Patterns
- Number Patterns
- Pascal Triangle
- Butterfly Patterns
- Concentric Number Patterns
- Alphabet Patterns

## How to Run

Clone the repository:

```bash
git clone https://github.com/Ankith-Achar-27/Pattern-Questions.git
```

Navigate to the project directory:

```bash
cd Pattern-Questions
```

Compile a Java file:

```bash
javac Pattern1.java
```

Run the program:

```bash
java Pattern1
```
