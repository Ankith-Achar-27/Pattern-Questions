# Pattern Questions in Java

A collection of classic pattern programming problems implemented in Java using nested loops. This repository is intended for beginners learning control flow, loops, and problem-solving techniques.


## Example

### Pyramid Pattern

```text
    *
   ***
  *****
 *******
*********
```

### Butterfly Pattern

```text
*        *
**      **
***    ***
****  ****
**********
****  ****
***    ***
**      **
*        *
```

## Concepts Used

- Nested Loops
- Conditional Statements
- Mathematical Relationships
- Pattern Observation
- Space and Character Manipulation

## How to Run

Clone the repository:

```bash
git clone https://github.com/Ankith-Achar-27/Pattern-Questions.git
```

Navigate to the project directory:

```bash
cd Pattern-Questions
```

Compile a Java file:

```bash
javac SolidSquare.java
```

Run the program:

```bash
java SolidSquare
```

## Learning Objective

Pattern problems help in understanding:

- Nested loops
- Loop boundaries
- Index manipulation
- Mathematical thinking
- Problem decomposition
